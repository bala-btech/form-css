# 📋 Student Registration Form

A simple, clean, and responsive web application featuring a structured **Student Registration Form** alongside a real-time data display layout. This project demonstrates semantic HTML5 layout structures styled entirely with customized, elegant CSS.

---

## 🚀 Features

* **User Input Fields:** Collects fundamental data points including *Full Name*, *Email*, *Password*, and *Date of Birth*.
* **Gender Selection:** Implements tailored radio buttons for mutually exclusive selection (*Male*, *Female*, *Other*).
* **Skills Checklist:** Fully customized checkboxes allowing users to select multiple technical skills (*HTML*, *CSS*, *JavaScript*).
* **Form Validation:** Built-in native HTML5 required attributes ensuring data integrity prior to processing.
* **Modern CSS Styling:** Form elements, typography, focus states, layouts, and data grids are beautifully enhanced with sleek CSS layouts.
* **Data Display Grid:** Includes a pre-styled, responsive tabular grid (`<table>`) featuring subtle zebra striping, hover configurations, and crisp borders to cleanly present mock or submitted records.

---

## 🛠️ Technologies Used

* **HTML5:** Semantic architecture, form input attributes, structural layouts.
* **CSS3:** Custom color variables, reset box-sizing model, responsive container grids, interactive button states, and polished tabular styling.

---

## 🎨 Visual Design Highlights

This project avoids generic browser styles by using a modern, cohesive UI theme:
* **Typography:** Accessible sans-serif system font stack for readability.
* **Visual Balance:** Balanced padding, clean borders, and smooth transitions on hover states.
* **Colors:** Deep professional tones with distinct interactive accent highlights.
